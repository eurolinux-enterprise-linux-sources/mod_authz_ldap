HOWTOs

While the configuration document tries to explain all the directives in detail, it cannot necessarily cover all possible uses of the module. Fortunately some users of the module have provided short step by step instructions or HOWTOs which we happily include here.

  • Beat Kneubuehl: MINI-HOWTO on how to set up mod_authz_ldap with the Apache webserver as a DSO module for x509 certificate verification and user mapping
  • Tim Tassonis: Example of application of mod_authz_ldap's use for authentication of a certificate against serial number (and issuer) and authorization against an authorizationRole attribute
